# Calculadora de Media

Projeto desenvolvido em Flutter para a disciplina de Desenvolvimento para Dispositivos Moveis (DPM).

## Autores
- Gabriel H.
- Pedro C.

## Funcionalidades
- Entrada de Nome e E-mail.
- Entrada de tres notas.
- Calculo automatico da media aritmetica.
- Interface para visualizacao dos resultados.
- Opcao para limpar todos os campos.

## Como executar
1. Certifique-se de ter o Flutter instalado em sua maquina.
2. Clone este repositorio.
3. Execute o comando 'flutter pub get' para instalar as dependencias.
4. Execute o projeto com 'flutter run'.
